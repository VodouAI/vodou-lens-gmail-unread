# vodou-lens-gmail-unread

A **Vodou Lens** that renders `gmail.unread`-matched URLs as a clean structured view inside a Vodou chat.

> Show the latest 10 unread emails from your Gmail inbox — sender, subject, snippet — by reading the page you're already logged into via Vodou Bridge.

## Install

```bash
vodou-core lenses install https://github.com/VodouAI/vodou-lens-gmail-unread
```

Or paste the URL into **+ Add lens** in the gateway sidebar's Lenses tab.
